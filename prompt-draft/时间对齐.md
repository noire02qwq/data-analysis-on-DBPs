# 时间对齐

cond-RT列是空列，直接去掉

标题中，前缀为测量的量，后缀为测量点名称。实验系统中，DT-RT-PPL1-PPL2是液体流动过程的四个测量点，所以同一批液体在不同测量点的对应时间是不同的。假如液体在t时从AT出发，流经测量点时间如下
- DT: t
- RT: t+25
- PPL1: t+40
- PPL2: t+55
请根据补全后的数据，输出时间对齐的新的表格：其中同一行代表同一批液体在每个测量点的测量值，时间列代表从AT的出发时间，也就是说对齐后时间为t的行DT RT PPL1 PPL2数据，对应对齐前的DTt RTt+25 PPL1t+40 PPL2t+55的数据。对齐后表格输出至data，并去除首尾数据不全的行
