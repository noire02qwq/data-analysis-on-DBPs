# 时间对齐

cond-RT列是空列，直接去掉

标题中，前缀为测量的量，后缀为测量点名称。实验系统中，DT-RT-PPL1-PPL2是液体流动过程的四个测量点，所以同一批液体在不同测量点的对应时间是不同的。假如液体在t时从AT出发，流经测量点时间如下

- DT: t
- RT: t+25h
- PPL1: t+38h
- PPL2: t+55h

比如5月16日12:00的DT，对应5月17日13:00的RT、5月18日2:00的PPL1、5月18日19:00的PPL2，把这一行数据输出到对齐后表格的5月16日12:00行

请根据补全后的数据，输出时间对齐的新的表格：其中同一行代表同一批液体在每个测量点的测量值，时间列代表从AT的出发时间，也就是说对齐后时间为t的行DT RT PPL1 PPL2数据，对应对齐前的DTt RTt+25h PPL1t+38h PPL2t+55h的数据。对齐后表格输出至data，并去除首尾数据不全的行
